# Explosive Sulfur Cube
Pretty self-explanatory. Feed the sulfur cube with TNT, let it crash & explode.
*Need to restart world upon first load or update.

- Sulfur cube with TNT
- Will explode if impact hard enough or upon death.
- Two modes: normal & incendiary
  - Normal: Same power & effect as TNT
  - Incendiary: Same power as TNT, but also spreads fire. Triggers when sulfur cube is on fire upon death/impact.
- Cubes have different impact thresholds when mounted, making it fun to use.
  - TNT sulfur cube can now ride boat
- Somewhat care for performance
  - Only those beyond certain speed will be included into motion calculation
